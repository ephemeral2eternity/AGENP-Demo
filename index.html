AGENS Overview

In commercial Video-on-Demand (VoD) systems, provisioning thousands of cache servers to serve millions of users with an expanding collection of movies is very common. As the system scales up, system management and control become great challenges. In this project, we propose AGENS --- an AGENt based management and control System for large-scale VoD. AGENS performs management and control in a large number of processes distributed all over the VoD system, referred to as agents. It inherently overcomes the difficulty of scalability via massive distributed agents. It also specializes agents to handle different types of tasks such as Quality of Experience (QoE) driven control and system management. Specifically, AGENS includes system management and control following aspects shown in Figure 1.

As it is shown in Figure 1, the AGENS system deploys agents in each server and client to monitor the health status of each server and user. All agents communicate with each other to exchange important information, make group decisions and organize themselves in a way to ensure the reachability of all distributed agents. With monitoring and management modules available, agents can make controls in the system to improve user Quality of Experience (QoE), higher availability of the system, and lower cost of resources, etc.

Objectives

  • Improve Quality of Experience for each individual user by adaptively change controls of content caching, resource provisioning and server selection.
  • Reduce the cost of resources by avoid over-provisioning resources.
  • Develop a fully distributed management and control system that adapts to various scales of system dynamically.
  • Improve the availability of VoD services by real-time monitoring and fail-over control.
